# Inspiration

## Why I am building on Base

Base is where the next billion users will come
onchain. Coinbase has the distribution, trust,
and resources to make it happen. Being an early
builder on Base positions you ahead of the curve.

## Builders who inspire me

### The Base ecosystem
The Base team ships fast and builds in public.
Watching how they operate taught me that
consistency beats perfection every time.

### Open source contributors
Every library this project uses was built by
someone who decided to share their work freely.
OpenZeppelin, wagmi, viem, OnchainKit — all open.

## What drives this project

### 100 commits goal
Not just a number. Each commit is proof of work.
Proof that you showed up, wrote code, and shipped.
The graph does not lie.

### Building the best chain
The bio says it — working hard to develop the
best chain. Every commit is a brick in that wall.

### Community over competition
The Base ecosystem grows when builders help each
other. Sharing this code openly means anyone
can learn from it and build something better.

## Lessons learned so far
- Start before you are ready
- Ship something every day
- Ask for help when stuck
- Document everything
- The terminal is your friend
- Errors are teachers not enemies
