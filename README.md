# Bhanks Base Project

Building on Base — ERC-20 token, NFT collection, staking protocol, and DAO.

## Tech stack
- [Base](https://base.org) — L2 blockchain
- [OnchainKit](https://onchainkit.xyz) — React components for Base
- [Next.js](https://nextjs.org) — Frontend framework
- [Hardhat](https://hardhat.org) — Smart contract development
- [wagmi](https://wagmi.sh) + [viem](https://viem.sh) — Wallet connection

## Status
In active development — building toward 100 public commits on Base
